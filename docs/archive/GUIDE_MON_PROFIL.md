# Guide: Gestion de Mon Profil

## Nouveauté: Section "Mon Profil" ajoutée ✅

Une nouvelle section **"Mon Profil"** a été ajoutée à la page des paramètres pour permettre à chaque utilisateur de gérer ses propres informations.

## Accès

1. Connectez-vous à l'application
2. Cliquez sur **Configuration** (⚙️) dans la navigation
3. Cliquez sur l'onglet **"Mon Profil"** (premier onglet)

## Fonctionnalités disponibles

### 1. Informations personnelles

Vous pouvez modifier:
- **Nom complet** - Votre nom d'affichage
- **Téléphone** - Votre numéro de téléphone
- **Email actuel** - Affiché en lecture seule (utilisez la section dédiée pour le changer)
- **Rôle** - Affiché en lecture seule (géré par un administrateur)

**Action:** Cliquez sur "Enregistrer les modifications" pour sauvegarder

### 2. Changer l'email

Pour changer votre adresse email:
1. Entrez votre **nouvelle adresse email**
2. Cliquez sur "Envoyer email de confirmation"
3. Vous recevrez un email de confirmation à la nouvelle adresse
4. Cliquez sur le lien dans l'email pour finaliser le changement

**Important:**
- Le changement ne sera effectif qu'après confirmation via email
- L'ancien email reste actif jusqu'à la confirmation
- Vous recevrez une notification sur les deux adresses

### 3. Changer le mot de passe

Pour changer votre mot de passe:
1. Entrez votre **mot de passe actuel** (requis pour vérification)
2. Entrez votre **nouveau mot de passe** (minimum 8 caractères)
3. **Confirmez** le nouveau mot de passe
4. Cliquez sur "Changer le mot de passe"

**Sécurité:**
- Le mot de passe actuel est vérifié avant tout changement
- Le nouveau mot de passe doit être différent de l'actuel
- Le changement prend effet immédiatement
- Vous devrez utiliser le nouveau mot de passe à la prochaine connexion

## Validation en temps réel

L'interface vous informe instantanément:
- ✅ Les mots de passe correspondent
- ❌ Les mots de passe ne correspondent pas
- ⚠️ Champs requis manquants
- ℹ️ Instructions et avertissements de sécurité

## Différence avec "Utilisateurs"

| Mon Profil | Utilisateurs (Admin) |
|------------|---------------------|
| Gérer **mes propres** informations | Gérer **les autres** utilisateurs |
| Changer **mon** email | Voir les emails des autres |
| Changer **mon** mot de passe | Réinitialiser les mots de passe des autres |
| Modifier **mon** nom et téléphone | Modifier les rôles et informations des autres |
| Accessible à **tous les utilisateurs** | Accessible uniquement aux **administrateurs** |

## Messages d'erreur courants

| Erreur | Solution |
|--------|----------|
| "Email invalide" | Vérifiez le format de l'email (doit contenir @) |
| "Le mot de passe doit contenir au moins 8 caractères" | Utilisez un mot de passe plus long |
| "Les mots de passe ne correspondent pas" | Retapez correctement le mot de passe |
| "Mot de passe actuel incorrect" | Vérifiez votre mot de passe actuel |
| "Le nouvel email est identique à l'actuel" | Utilisez une adresse différente |

## Support

Si vous rencontrez des problèmes:
1. Vérifiez que vous êtes connecté
2. Actualisez la page (Ctrl+R ou Cmd+R)
3. Vérifiez votre connexion internet
4. Contactez votre administrateur si le problème persiste

## Sécurité et confidentialité

- ✅ Toutes les modifications nécessitent une authentification
- ✅ Le changement d'email nécessite une confirmation par email
- ✅ Le changement de mot de passe nécessite le mot de passe actuel
- ✅ Vos informations sont protégées par chiffrement
- ✅ Seuls vous et les administrateurs peuvent voir vos informations complètes
